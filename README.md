# DHL Network Design Simulation

##  Overview
This project simulates a real-world network infrastructure for DHL, a global logistics and delivery company. The network is designed to support efficient communication across multiple international locations using Cisco Packet Tracer.

The system connects three locations:
- Germany
- Australia
- Saudi Arabia

Each location has its own LAN while remaining fully connected to the global network.

---

## Features
- Multi-location network (3 LANs)
- 9 PCs, 3 laptops, 3 printers
- 14 routers, 3 switches, 1 DHCP server
- Dynamic routing using RIP protocol
- Automatic IP assignment (DHCP)
- Full communication between all networks

## 🌐 How It Works
1. Each location has its own LAN  
2. Devices connect using switches  
3. Routers connect all LANs together  
4. RIP protocol manages routing  
5. DHCP assigns IP addresses automatically  
6. All devices can communicate across networks
